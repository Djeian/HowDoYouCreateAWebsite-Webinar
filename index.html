What is a website?

A hosted server through the internet to provide user entertainment, information, and selling of products

What is a website composed of?

A website is made up of Markup Programming Languages and Programming Languages.

What is a Markup Programming Language?

A Markup Programming Language describe what will be the output of the content, it uses tags or elements to build the structure of the website, it is the skeletal system of the website.

What is a Programming Language?

It is use to communicate with the computer to assign task with the use of Programming Languages, because a computer has its own languages like humans do.
